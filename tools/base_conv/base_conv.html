2sC Base Conversion

Type in a valid 16 bit 2sC value into any input field and hit Enter or click the Calculate button. The output will update. Valid decimal values are -32768 to 32767. Valid binary is 16 bits of 0s and 1s. Valid hexadecimal is 4 nibbles, each 0-9 or A-F (lowercase acceptable).

The status label below will inform you if an invalid input has been entered.

Input Output
Decimal: # Decimal: #
Binary: b Binary: b
Hexadecimal: x Hexadecimal: x
